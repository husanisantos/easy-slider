== Description ==

Easy Slider is a WordPress plugin that allows you to easily create image sliders on your website. It is compatible with WordPress version 6 or later and is licensed under the GNU General Public License version 3.0.

Flexible layout options including swipe direction, transition speed and repeat interval
Support for image captions and descriptions
Integration with the Gutenberg Block Editor

== Installation ==
 
1. In the Plugins menu of the Wordpress Panel click on Add New
2. Upload eslider.zip
